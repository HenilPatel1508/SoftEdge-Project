# 🛒 MERN E-Commerce Admin + User Dashboard

A full-stack eCommerce web application built using **MERN Stack (MongoDB, Express, React, Node.js)** with modern UI, authentication, admin dashboard, order management, and payment flow.

---

## 🚀 Features

### 👤 User Side
- User registration & login (JWT auth)
- Product browsing & filtering
- Add to cart / update quantity
- Checkout & order placement
- Order history
- Invoice download

### 🛠️ Admin Panel
- Dashboard with sales analytics
- Add / Edit / Delete products
- Manage users
- Manage orders
- View sales chart (Recharts)
- Delete orders

---

## 🧑‍💻 Tech Stack

### Frontend
- React.js (Vite)
- Redux Toolkit
- Tailwind CSS
- ShadCN UI
- Axios
- Recharts
- Sonner Toast

### Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT Authentication
- Multer (Image upload)

### Deployment
- Frontend: Render / Vercel
- Backend: Render
- Database: MongoDB Atlas
